# Image Indexing
Operating System : Ubuntu 18.04.3 LTS

Library dijalankan di Anaconda3 Distribution dan Python versi 3.7.3

## Petunjuk Penggunaan
1. Clone repository menggunakan perintah pada CLI :
    ```bash
    $ git clone https://github.com/ChiyenLee/py-image.git
    ```
